# TravelTracker - Планировщик путешествий с учётом расходов

## Описание проекта
TravelTracker - это настольное приложение для планирования путешествий и учёта расходов во время поездок. Приложение позволяет контролировать бюджет, анализировать расходы по категориям и получать рекомендации для экономии.

## Основные функции
1. **Управление путешествиями**
   - Создание и редактирование поездок
   - Установка бюджета и сроков
   - Отслеживание статуса поездки

2. **Учёт расходов**
   - Добавление расходов по категориям
   - Привязка расходов к конкретным поездкам
   - Указание способа оплаты и места расхода

3. **Анализ данных**
   - Визуализация расходов по категориям
   - Сравнение бюджета с фактическими расходами
   - Анализ дневных расходов
   - Генерация рекомендаций

4. **Работа с внешними API**
   - Получение курсов валют
   - Конвертация сумм

5. **Импорт/Экспорт**
   - Экспорт данных в CSV и JSON
   - Сохранение графиков в формате PNG
